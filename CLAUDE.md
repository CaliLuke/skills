# Skills Repository

This is a collection of agent skills. Each skill is in `skills/{skill-name}/SKILL.md`.

## Available Skills

- **typedb** - TypeQL language reference for TypeDB 3.8+. Schema definition, CRUD, queries, functions, and pitfalls.

## Usage

Skills are loaded on-demand based on their description. When a task matches a skill's description, the full SKILL.md is loaded into context.

## Adding Skills

Copy the `template/` folder, rename it, and edit the SKILL.md with your instructions.
